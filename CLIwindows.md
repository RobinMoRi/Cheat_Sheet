## Environmental Variables
```sh
set #Displays all the environmental variables
set varname #Display the value of the variable varname
set varname=value #Set or change the valeu of varname
```

## Basic Commands
```sh

dir #ls

findstr name #grep
  -i: not case-sensitive
  
type file.txt #cat
```
